# Replace text

**Java - Android**
```
result = text.replace("from", "to");
```

**Javascript**
```
result = text.replace(/from/g, 'to');
```

**Objective-C**
```
result = [text stringByReplacingOccurrencesOfString:@"from" withString:@"to"];
```

**PHP**
```
$result = str_replace('from', 'to', $text);
``````
