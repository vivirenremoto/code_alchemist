# Concat two strings

**Java - Android**
```
result = "Hello " + "world";
```

**Javascript**
```
result = "Hello " + "world";
```

**Objective-C**
```
NSString *string1 = @"Hello ";
NSString *string2 = @"world";
NSString *result = [NSString stringWithFormat:@"%@%@", string1, string2];
```

**PHP**
```
$result = "Hello " . "world";
```
